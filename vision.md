# Home Intelligence

Version: 0.1.0

## Purpose

Home Intelligence is a long-term project to improve the comfort,
efficiency, observability, and automation of our home.

The project is intentionally data-driven.

Rather than automating devices first and measuring later, the project
emphasizes collecting telemetry, understanding system behavior, and only
then implementing automations based upon measured results.

The system should answer questions such as:

- Where is our electricity actually going?
- What effect do changes to HVAC operation have?
- Which automations genuinely save money?
- What actions improve comfort?
- How would alternative Rocky Mountain Power rate schedules affect us?

The project should also provide a foundation for future AI-driven home
optimization.
